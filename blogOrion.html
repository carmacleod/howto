How to write a Blog for Orion

  1. Log in to PlanetOrion using Orion credentials (with strong password).
  2. Click on "Add New", and give it a title and a category.
  3. Use the Wordpress editor. You can edit visually on the Visual tab and/or edit the HTML on the Text tab.
  4. There is an "add media" button for inserting images.
  5. Make sure you just "save draft" instead of "publish" until you are completely done. It can be in draft form as long as needed.
